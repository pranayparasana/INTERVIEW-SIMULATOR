# INTERVIEW-SIMULATOR
# 🎯 AI Interview Simulator

AI-powered mock interview platform that simulates real interviews,
asks role-specific questions, evaluates answers and generates scorecards.

## Roles Supported
Product Manager · Data Analyst · Business Analyst · Software Engineer · AI/ML Engineer

## Tech Stack
Python · Groq LLaMA3 · Streamlit

## How to Run
1. Clone this repo
2. pip install -r requirements.txt
3. Add your Groq API key to .env file: GROQ_API_KEY=your-key
4. streamlit run app.py
